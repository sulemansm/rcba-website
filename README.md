# Rotaract Club of Bombay Airport — Website

## Structure
```
rcba-site/
├── index.html              ← Main website (standalone, works anywhere)
├── public/
│   ├── admin/
│   │   ├── config.yml      ← Decap CMS configuration
│   │   └── index.html      ← CMS admin panel
│   └── images/
│       └── rcba_logo.png   ← Club logo
├── content/                ← Auto-generated by CMS (after setup)
│   ├── projects/
│   ├── gallery/
│   ├── impact/
│   ├── testimonials/
│   └── settings/
└── README.md
```

## CMS Setup (Forever Free with Netlify)

1. **Push this folder to GitHub** (make it public or private)
2. **Connect to Netlify** → New Site from Git → select your repo
3. **Enable Identity** → Site Settings → Identity → Enable
4. **Enable Git Gateway** → Identity → Services → Git Gateway → Enable
5. **Invite yourself** → Identity → Invite users → your email
6. Visit `yoursite.netlify.app/admin` → log in → start editing!

## What you can manage via CMS
- ✅ Projects (add/edit/delete, reorder, toggle active)
- ✅ Gallery items (emoji, caption, size, color theme)  
- ✅ Impact numbers (change counts)
- ✅ Testimonials (add member stories)
- ✅ Site settings (hero text, contact info, social links)

## Alternative free hosts
- **Cloudflare Pages** → same setup, even faster CDN
- **GitHub Pages** → use Netlify Identity Widget with external auth

## Tech Stack
- Pure HTML/CSS/JS — no build step needed
- Plus Jakarta Sans (Google Fonts)
